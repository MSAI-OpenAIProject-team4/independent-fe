# independent-fe

MSAI OpenAI 프로젝트 프론트엔드입니다.

## 프로젝트 소개
독립운동가와의 대화를 통해 역사를 배우고, AI 얼굴 매칭 기술을 활용하여 사용자와 닮은 독립운동가를 찾아주는 웹 애플리케이션입니다.

## 주요 기능
- 독립운동가와의 대화형 인터페이스
- 사용자 얼굴 촬영 및 독립운동가 매칭
- 다국어 지원 (한국어, 영어, 일본어)

## 기술 스택
- React
- React Router
- CSS3
- OpenAI API

## 설치 및 실행
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start
```
