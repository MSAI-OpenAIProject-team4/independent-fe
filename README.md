# independent-fe

MSAI OpenAI 프로젝트 프론트엔드입니다.

## 프로젝트 소개
독립운동가와의 대화를 통해 역사를 배우고, AI 얼굴 매칭 기술을 활용하여 사용자와 닮은 독립운동가를 찾아주는 웹 애플리케이션입니다.

## 주요 기능
- 독립운동가와의 대화형 인터페이스
- 사용자 얼굴 촬영 및 독립운동가 매칭
- 다국어 지원 (한국어, 영어, 일본어)

## 실행 방법

### 1. Python 서버 실행
```bash
# 서버 디렉토리로 이동
cd server

# 필요한 패키지 설치
pip install -r requirements.txt

# 서버 실행
python app.py
```
서버가 성공적으로 실행되면 http://localhost:5000 에서 실행됩니다.

### 2. React 앱 실행
```bash
# 프로젝트 디렉토리에서
npm install
npm start
```
